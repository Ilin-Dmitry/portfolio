01

Russian Travel

This project is about traveling around Russia, including in a quite unusual way - by commuter trains. It contains links to useful resources that can help while organizing a trip - maps, railway transport schedules, accommodation booking service, etc.

This is an adaptive website with grid changes at four breakpoints. It is created using flexbox and grid. The layout was made according to the BEM standard (Nested).
